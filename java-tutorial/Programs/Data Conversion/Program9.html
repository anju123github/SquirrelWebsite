9. How to convert String to Date

Code


                
import java.text.ParseException;
import java.text.SimpleDateFormat;
import java.util.Date;
                                
public class StringToDateConverter {
                                
    public static void main(String[] args) {
        // Example string containing a date
        String dateString = "2023-11-14";
                                
        // Define the date format to match the pattern in the string
        SimpleDateFormat dateFormat = new SimpleDateFormat("yyyy-MM-dd");
                                
        try {
            // Parse the string to a Date object using the defined format
            Date convertedDate = dateFormat.parse(dateString);
                                
            // Print the result
            System.out.println("Converted date: " + convertedDate);
        } catch (ParseException e) {
            // Handle the exception (e.g., print an error message)
            System.out.println("Error: Unable to convert the string to a date.");
        }
    }
}

Output

    Converted date: Mon Nov 14 00:00:00 UTC 2023
