13. Write a Java method that checks whether all the characters in a given string are vowels (a, e,i,o,u) or not. Return true if each character in the string is a vowel, otherwise return false.

Code


                public class VowelChecker {

                    // Method to check if all characters in a string are vowels
                    public static boolean areAllCharactersVowels(String input) {
                        // Convert the string to lowercase for case-insensitive comparison
                        String lowerCaseInput = input.toLowerCase();
                                                
                        // Check if each character in the string is a vowel
                        for (char ch : lowerCaseInput.toCharArray()) {
                            if (ch != 'a' && ch != 'e' && ch != 'i' && ch != 'o' && ch != 'u') {
                                return false; // If any character is not a vowel, return false
                            }
                        }
                        return true; // All characters are vowels
                    }
                                                
                    public static void main(String[] args) {
                        // Example usage
                        String str1 = "aeiou";
                        String str2 = "hello";
                                                
                        System.out.println("Are all characters in \"" + str1 + "\" vowels? " + areAllCharactersVowels(str1));
                        System.out.println("Are all characters in \"" + str2 + "\" vowels? " + areAllCharactersVowels(str2));
                    }
                }

Output

Are all characters in "aeiou" vowels? true 

Are all characters in "hello" vowels? false
