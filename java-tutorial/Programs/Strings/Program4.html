4. Program to remove all occurrences of a given character from input String?

Code


        
import java.util.Scanner;
public class RemoveCharacterFromString {
    public static void main(String[] args) {
        // Get input from the user
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter a string: ");
        String inputString = scanner.nextLine();

        // Get the character to be removed
        System.out.print("Enter the character to remove: ");
        char charToRemove = scanner.next().charAt(0);

        // Remove all occurrences of the given character
        String resultString = removeCharacter(inputString, charToRemove);

        // Display the original and modified strings
        System.out.println("Original String: " + inputString);
        System.out.println("String after removing '" + charToRemove + "': " + resultString);
    }

    // Function to remove all occurrences of a character from a string
    private static String removeCharacter(String inputString, char charToRemove) {
        // Using replaceAll to remove all occurrences of the character
        return inputString.replaceAll(String.valueOf(charToRemove), "");
   }
}

Output

    Enter a string: programming is fun
    Enter the character to remove: g
    Original String: programming is fun
    String after removing 'g': proammin is fun
