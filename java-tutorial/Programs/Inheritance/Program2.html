Inheritance

2. Write a Java program to create a class called Vehicle with a method called drive(). Create a subclass called Car that overrides the drive() method to print "Repairing a car".

Code


                
// Base class Vehicle
class Vehicle {
    // Method to drive the vehicle
    public void drive() {
        System.out.println("Vehicle is being driven");
    }
}
                                        
// Subclass Car extending Vehicle
class Car extends Vehicle {
    // Override the drive method to print "Repairing a car"
    
    public void drive() {
        System.out.println("Repairing a car");
    }
}
                                        
// Main class
public class VehicleExample {
    public static void main(String[] args) {
        // Create an instance of Vehicle
        Vehicle myVehicle = new Vehicle();

        // Create an instance of Car
        Car myCar = new Car();

        // Demonstrate drive for both instances
        myVehicle.drive(); // Output: Vehicle is being driven
        myCar.drive();     // Output: Repairing a car
    }
}

Output

Vehicle is being driven 

Repairing a car
