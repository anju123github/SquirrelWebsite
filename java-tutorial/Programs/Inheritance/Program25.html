Inheritance

25. Write a Java program to create an abstract class Bird with abstract methods fly() and makeSound(). Create subclasses Eagle and Hawk that extend the Bird class and implement the respective methods to describe how each bird flies and makes a sound.

code


             
// Abstract class Bird
abstract class Bird {
    // Abstract methods
    public abstract void fly();
    public abstract void makeSound();
}
                                        
// Subclass Eagle extending Bird
class Eagle extends Bird {
    // Implementing the fly method for Eagle
    
    public void fly() {
        System.out.println("Eagle soars high in the sky");
    }
                                        
    // Implementing the makeSound method for Eagle
    
    public void makeSound() {
        System.out.println("Eagle screeches loudly");
    }
}
                                        
// Subclass Hawk extending Bird
class Hawk extends Bird {
    // Implementing the fly method for Hawk
    
    public void fly() {
        System.out.println("Hawk glides gracefully through the air");
    }
                                        
    // Implementing the makeSound method for Hawk
    
    public void makeSound() {
        System.out.println("Hawk emits a sharp cry");
    }
}
                                        
// Main class
public class AbstractBirdExample {
    public static void main(String[] args) {
        // Create instances of Eagle and Hawk
        Eagle eagle = new Eagle();
        Hawk hawk = new Hawk();
                                        
        // Describe how each bird flies and makes a sound
        System.out.println("Eagle:");
        eagle.fly();
        eagle.makeSound();
                                        
        System.out.println(); // Empty line for clarity
                                        
        System.out.println("Hawk:");
        hawk.fly();
        hawk.makeSound();
    }
}

Output
