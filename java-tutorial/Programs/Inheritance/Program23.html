Inheritance

23. Write a Java program to create an abstract class Instrument with abstract methods play() and tune(). Create subclasses for Glockenspiel and Violin that extend the Instrument class and implement the respective methods to play and tune each instrument.

Code


                
// Abstract class Instrument
abstract class Instrument {
    // Abstract methods
    public abstract void play();
    public abstract void tune();
}
                                        
// Subclass Glockenspiel extending Instrument
class Glockenspiel extends Instrument {
    // Implementing the play method for Glockenspiel
    
    public void play() {
        System.out.println("Glockenspiel plays melodious tunes");
    }
                                        
    // Implementing the tune method for Glockenspiel
    
    public void tune() {
        System.out.println("Glockenspiel is tuned by adjusting metal bars");
    }
}
                                        
// Subclass Violin extending Instrument
class Violin extends Instrument {
    // Implementing the play method for Violin
    
    public void play() {
        System.out.println("Violin produces beautiful string sounds");
    }
                                        
    // Implementing the tune method for Violin
    
    public void tune() {
        System.out.println("Violin is tuned by adjusting the tension of the strings");
    }
}
                                        
// Main class
public class AbstractInstrumentExample {
    public static void main(String[] args) {
        // Create instances of Glockenspiel and Violin
        Glockenspiel glockenspiel = new Glockenspiel();
        Violin violin = new Violin();
                                        
        // Play and tune the instruments
        System.out.println("Glockenspiel:");
        glockenspiel.play();
        glockenspiel.tune();
                                        
        System.out.println(); // Empty line for clarity
                                        
        System.out.println("Violin:");
        violin.play();
        violin.tune();
    }
}

Output
